# Shift Keying Encoder

Shift-keying is a method used widely for data transmission over things such as phone lines, radio waves, et c. One quite famous example is the the Bell 202 modem. This script generators PSK, FSK, or ASK (configurably) depending on the configuration.

# Important Note
This script was adapted from a SAME header encoder [located here and made by nicksmadscience.](https://github.com/nicksmadscience/eas-same-encoder/). This script is a generalized version of that one.
